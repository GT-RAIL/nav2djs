Class: ImageMapClientNav

NAV2D. ImageMapClientNav

new ImageMapClientNav(options)

A OccupancyGridClientNav uses an OccupancyGridClient to create a map for use with a Navigator.
Parameters:
Name Type Description
options object with following keys: * ros - the ROSLIB.Ros connection handle * topic (optional) - the map meta data topic to listen to * image - the URL of the image to render * serverName (optional) - the action server name to use for navigation, like '/move_base' * actionName (optional) - the navigation action name, like 'move_base_msgs/MoveBaseAction' * rootObject (optional) - the root object to add the click listeners to and render robot markers to * withOrientation (optional) - if the Navigator should consider the robot orientation (default: false) * viewer - the main viewer to render to
